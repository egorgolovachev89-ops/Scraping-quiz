Вопросы и варианты ответов лежат прямо в HTML этой страницы. Попробуй извлечь их через requests + BeautifulSoup:

import requests from bs4 import BeautifulSoup url = 'PASTE_THIS_PAGE_URL_HERE' response = requests.get(url) soup = BeautifulSoup(response.text, 'html.parser') # вопросы хранятся в JS-массиве QUIZ_DATA # попробуй найти его внутри тега <script> script = soup.find('script', string=lambda s: s and 'QUIZ_DATA' in s) # ...а дальше — твоя задача :)

💡 Подсказка: данные квиза находятся в переменной QUIZ_DATA внутри тега <script>. Извлеки строку, найди JSON через регулярку или json.loads, и получишь все вопросы программно.
